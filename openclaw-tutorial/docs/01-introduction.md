# 第一章：OpenClaw 简介

## 什么是 OpenClaw？

OpenClaw 是一个强大的 AI 助手框架，它可以帮你自动化各种任务。与普通的聊天机器人不同，OpenClaw 可以：

- 访问你的文件系统
- 执行 shell 命令
- 操作飞书、GitHub 等第三方服务
- 运行定时任务（类似 cron）
- 控制浏览器

## OpenClaw 能做什么？

想象一下，有一个 AI 助手可以帮你：

1. **多渠道消息** - 通过 WhatsApp、Telegram、Discord、iMessage 等聊天
2. **管理日程** - 自动检查日历、发送提醒
3. **处理邮件** - 阅读、分类、甚至回复邮件
4. **操作文件** - 创建、编辑、整理文件
5. **运行代码** - 执行开发任务、写代码
6. **定时任务** - 每天定时做某些事情
7. **控制浏览器** - 自动化网页操作
8. **移动节点** - 配对 iOS/Android 设备拍照、截屏、语音
