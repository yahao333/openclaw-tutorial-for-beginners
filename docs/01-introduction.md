# 第一章：OpenClaw 简介

## 什么是 OpenClaw？

OpenClaw 是一个强大的 AI 助手框架，它可以帮你自动化各种任务。与普通的聊天机器人不同，OpenClaw 可以：

- 访问你的文件系统
- 执行 shell 命令
- 操作飞书、GitHub 等第三方服务
- 运行定时任务（类似 cron）
- 控制浏览器

## OpenClaw 能做什么？

想象一下，有一个 AI 助手可以帮你：

1. **管理日程** - 自动检查日历、发送提醒
2. **处理邮件** - 阅读、分类、甚至回复邮件
3. **操作文件** - 创建、编辑、整理文件
4. **运行代码** - 执行开发任务、写代码
5. **定时任务** - 每天定时做某些事情
