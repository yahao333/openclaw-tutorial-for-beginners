# 第十章：日历管理

## 概述

OpenClaw 可以与你的日历集成，自动管理日程安排。

## 支持的日历

- Google Calendar
- 飞书日历
- Outlook Calendar
- Apple Calendar（通过 CalDAV）

## 基本功能

### 查看日程

```
今天有什么安排？
明天上午10点有会吗？
```

### 创建日程

```
帮我创建一个会议：
- 主题：项目评审
- 时间：明天下午3点
- 参会人：zhangsan@example.com
```

### 修改日程

```
把明天的会议改到下午4点
```

### 删除日程

```
取消明天下午3点的会议
```

## 高级功能

### 1. 自动提醒

在事件发生前自动提醒你：

```markdown
HEARTBEAT.md 配置：
- 日历：提前30分钟提醒
- 会议：提前15分钟提醒
```

### 2. 智能日程分析

分析你的日历习惯：

- 找出空闲时间
- 识别忙碌模式
- 建议最佳会议时间

### 3. 自动处理邀请

- 自动接受确定的会议
- 拒绝冲突的会议
- 提议新时间

### 4. 日报/周报生成

自动生成日程汇总：

```
今天/本周工作总结：
- 3个会议
- 2小时空闲时间
- 需要准备的资料：...
```

## 集成飞书日历

如果使用飞书，可以直接通过飞书 API 管理日历：

```json
{
  "feishu": {
    "calendar": {
      "enabled": true
    }
  }
}
```
