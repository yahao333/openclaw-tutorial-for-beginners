# 第九章：邮件自动化

## 概述

OpenClaw 可以帮你自动处理邮件，包括读取、分类、发送邮件等功能。

## 配置邮件服务

### Gmail 配置

1. 开启 IMAP 访问
2. 生成应用专用密码
3. 在配置文件中添加：

```json
{
  "email": {
    "provider": "gmail",
    "username": "your-email@gmail.com",
    "password": "your-app-password"
  }
}
```

## 使用场景

### 1. 读取未读邮件

```
帮我看看有没有重要的未读邮件
```

### 2. 邮件分类

自动将邮件分类到不同的标签下：

- 工作邮件 → "工作"标签
- 促销邮件 → "促销"标签
- 重要邮件 → 标记星标

### 3. 自动回复

设置自动回复规则：

```markdown
当收到邮件主题包含 "请假" 时：
自动回复批准模板
```

### 4. 定时发送邮件

在指定时间自动发送邮件：

```
每天下午5点发送日报汇总
```

## 安全注意事项

- 使用应用专用密码，不要使用登录密码
- 定期检查邮件访问权限
- 启用双因素认证
