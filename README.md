# OpenClaw 教程

一个面向小白的 OpenClaw 入门教程。

## 教程大纲

本教程共分为 14 个章节，帮助你从零开始掌握 OpenClaw：

1. [第一章：OpenClaw 简介](./docs/01-introduction.md) - 了解 OpenClaw 是什么
2. [第二章：环境准备](./docs/02-environment-setup.md) - 安装必要的工具
3. [第三章：初次运行](./docs/03-first-run.md) - 启动你的第一个 OpenClaw
4. [第四章：基础配置](./docs/04-basic-configuration.md) - 配置你的 OpenClaw
5. [第五章：技能使用](./docs/05-using-skills.md) - 使用 OpenClaw 的技能系统
6. [第六章：飞书集成](./docs/06-feishu-integration.md) - 连接飞书
7. [第七章：自定义技能](./docs/07-custom-skills.md) - 开发自己的技能
8. [第八章：高级主题](./docs/08-advanced-topics.md) - 深入了解 OpenClaw
9. [第九章：邮件自动化](./docs/09-email-automation.md) - 自动处理邮件
10. [第十章：日历管理](./docs/10-calendar-management.md) - 智能日程管理
11. [第十一章：社交媒体集成](./docs/11-social-media.md) - Discord、Slack、Twitter
12. [第十二章：AI 能力集成](./docs/12-ai-capabilities.md) - 图像生成、语音合成
13. [第十三章：数据库操作](./docs/13-database-operations.md) - MySQL、MongoDB、Redis
14. [第十四章：文件自动化](./docs/14-file-automation.md) - PDF、Excel、图片处理

## 快速开始

```bash
# 克隆仓库
git clone https://github.com/yahao333/openclaw-tutorial-for-beginners.git

# 查看教程
cd openclaw-tutorial-for-beginners/docs
```
